WEB PROJECT

Web Project ini berupa sebuah website khusus komunitas AMV Indonesia. Untuk apa adanya Website ? Landasan dasarnya karena kami ingin membuat AMVIndo lebih dikenal dan menjadi sarana informasi. Sosial Media utama kami yaitu Facebook & Youtube, dan kami merasa perlu adanya fitur tambahan yang dapat mempermudah komunitas ini agar dapat berjalan semestinya bahkan lebih baik lagi. Dengan adanya website, diharapkan dapat melengkapi kekurangan yang ada dan mempermudah kita semua.
